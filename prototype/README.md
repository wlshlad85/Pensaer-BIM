# Pensaer Prototype

This folder contains the working HTML prototype of Pensaer.

## Files

- `pensaer-prototype.html` - The main prototype file (to be added)

## Running the Prototype

Simply open the HTML file in a modern browser:

```bash
# macOS
open pensaer-prototype.html

# Windows
start pensaer-prototype.html

# Linux
xdg-open pensaer-prototype.html
```

## Features in Prototype

- ✅ 2D canvas with SVG rendering
- ✅ 3D view with Three.js
- ✅ Command palette (⌘K)
- ✅ Context menus
- ✅ AI suggestions
- ✅ Issue detection
- ✅ Relationship model
- ✅ Snap system
- ✅ Undo/redo
