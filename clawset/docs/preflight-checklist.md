# Clawhatch Pre-Flight Checklist

Send this to the client BEFORE your setup session.

---

## Before We Start — Please Have Ready:

### Required
- [ ] **Computer** — Mac, Linux, or Windows (WSL2 required for Windows)
- [ ] **Node.js 22+** installed — check with `node --version`
- [ ] **AI subscription** — one of:
  - Anthropic API key (recommended) — get at console.anthropic.com
  - Claude Pro/Max subscription (for OAuth)
  - OpenAI API key or ChatGPT Plus (Codex)
- [ ] **30 minutes** of uninterrupted time

### For WhatsApp Setup (optional but recommended)
- [ ] **Second phone number** — SIM, eSIM, or prepaid
  - This will be your assistant's WhatsApp number
  - WhatsApp installed and verified on that number
  - Phone nearby for QR scan during setup

### For Telegram Setup (optional)
- [ ] Create a bot via @BotFather on Telegram
- [ ] Save the bot token

### For Discord Setup (optional)
- [ ] Discord Developer Portal access
- [ ] Bot token ready (or we'll create one together)

### Nice to Have
- [ ] **Brave Search API key** — for web search capability
  - Free tier: brave.com/search/api
- [ ] A name for your assistant (we'll personalize it!)

---

## Windows Users — Important!

OpenClaw works best on WSL2 (Windows Subsystem for Linux). If you don't have it:

```powershell
wsl --install
```

Restart your computer, then we'll do the setup inside WSL2.
