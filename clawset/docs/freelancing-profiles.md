### FAQs

**Q: Do I need my own server?**
A: Yes ‚Äî OpenClaw needs to run somewhere. A $5-10/month VPS (DigitalOcean, Hetzner, Railway) works great. I can recommend the best option for your budget and help you set it up.

**Q: How much will the AI cost me monthly after setup?**
A: Depends on usage. Light use: $60-90/month. Moderate: $180-240/month. I configure smart model routing to minimize costs ‚Äî using cheaper models for simple tasks and premium models only when needed.

**Q: Can you set this up on Windows?**
A: Yes! Windows requires WSL2, which I'll configure for you. It's one of the trickiest parts of setup and a common reason people get stuck.

**Q: What if something breaks after delivery?**
A: Every package includes follow-up support (3-14 days depending on tier). I also offer ongoing monthly support for $29/month if you want peace of mind long-term.

**Q: Is this against Anthropic's Terms of Service?**
A: OpenClaw uses the official API with your own API key ‚Äî this is fully within Anthropic's TOS. I do NOT set up MAX subscription workarounds.

**Q: Can I see it working before I pay?**
A: Message me and I'll share screenshots/demos from previous setups (anonymized). I also don't mark the order complete until you confirm it's working.

**Q: What channels can you connect?**
A: WhatsApp, Telegram, Discord, Slack, SMS (Twilio), email, and webchat. The Standard and Premium tiers include multiple channels.
