**Developer tools:** Dark backgrounds (#0a0a0a to #1a1a2e), neon accents (cyan, purple, green), high contrast, monospace typography. Examples: Vercel (black/white), Railway (purple gradients), Linear (purple/blue).

## 3. Logo Concepts

### Concept 1: "The Claw Mark"
- **Looks like:** Three curved slash marks (like claw scratches) forming a subtle "C" shape. Minimal, geometric.
- **Communicates:** Speed, power, precision. "We got this handled."
- **Style:** Flat icon, works at any size. Think Linear's mark or Stripe's "S."

### Concept 2: "The Lobster Setup"
- **Looks like:** A stylized lobster claw holding/gripping a gear or terminal cursor (‚ñå). The claw is simplified to ~5 geometric shapes.
- **Communicates:** AI assistant that builds and configures. Friendly but capable.
- **Style:** Flat mascot-icon hybrid. Like GitHub's Octocat simplified to icon form.

### Concept 3: "Shell Bracket"
- **Looks like:** A lobster claw shape that doubles as code brackets `{ }` or angle brackets `< >`. The negative space between two claw pincers forms the bracket.
- **Communicates:** Code-native. Developer tool. Claw = code.
- **Style:** Geometric wordmark icon. Pairs with "Clawhatch" text.

### Concept 4: "The Snap"
- **Looks like:** Two claw pincers about to snap together, with a small spark/starburst at the contact point. Implies "snap and it's done."
- **Communicates:** Instant, effortless setup. One snap = configured.
- **Style:** Flat with a single accent color spark. Motion implied through the gap.

### Concept 5: "The Closet Door" ‚≠ê BEST FIT FOR "CLAWHATCH"
- **Looks like:** A slightly open door/cabinet silhouette with a lobster claw peeking out ‚Äî or the door handle IS a claw. Plays on the closet/clawhatch pun directly.
- **Communicates:** "Open your Clawhatch" = open your toolkit. Whimsy + utility. Discovery, possibility.
- **Style:** Flat icon with subtle depth (door shadow). Works as app icon perfectly. Rounded, friendly.

## 4. UX Inspiration

### Best-in-Class Setup Wizards

**Vercel's `create-next-app` & Dashboard Onboarding**
- Terminal-first with beautiful prompts (colored text, spinners)
- Dashboard wizard: 3-step import (connect repo ‚Üí configure ‚Üí deploy)
- Progress shown as a deploy log streaming in real-time
- ‚úÖ Celebration: confetti animation on first successful deploy

**Railway's Canvas Onboarding**
- Visual graph/canvas showing your infrastructure
- Drag-and-drop service creation
- Auto-detection of project type with magic configuration
- Progress feels organic ‚Äî things "appear" on your canvas

**Stripe's Developer Onboarding**
- Step-by-step with live code preview
- Toggle between languages (curl, Node, Python, etc.)
- "Test mode" green banner ‚Äî safe to experiment
- Checklist sidebar showing completion progress

**Linear's Setup Flow**
- Dark, minimal, fast
- Import from Jira/Asana with progress bar
- Workspace customization feels instant
- Keyboard shortcuts taught inline during setup

**Supabase Project Creation**
- Region selection on a world map
- Database provisioning with animated progress
- Instant API keys shown with copy buttons
- Guided "Quick Start" cards after creation

### UI Patterns to Steal

| Pattern | Best Example | Why It Works |
|---------|-------------|--------------|
| **Stepper with icons** | Stripe onboarding | Each step has a unique icon, completed steps get a green check |

| **Headlines** | **Inter** (Bold/Black) | 700-900 | Already in OpenClaw ecosystem. Clean, geometric, scales beautifully. |

## 6. Competitor Visual Audit

### Vercel
- **Colors:** Black, white, minimal. Almost no color ‚Äî relies on layout and typography.
- **Vibe:** Premium, minimal, "we're serious engineers." Swiss design influence.
- **Logo:** Simple triangle/play button. Geometric.
- **Differentiator:** Extreme minimalism. White space as a design element.

### Railway
- **Colors:** Purple-heavy gradients, dark backgrounds, bright accents.
- **Vibe:** Fun, visual, "infrastructure should be beautiful." Younger energy.
- **Logo:** Stylized "R" mark. Purple-forward.
- **Differentiator:** Visual canvas metaphor. Infrastructure as art.

### Linear
- **Colors:** Purple-blue gradients on dark. Minimal but with color personality.
- **Vibe:** Fast, precise, opinionated. "We care about craft."
- **Logo:** Geometric lines forming an "L." Animated gradient fills.
- **Differentiator:** Speed as brand identity. Everything feels instant.

### Stripe
- **Colors:** Purple primary (#635BFF), vibrant gradients, light backgrounds.
- **Vibe:** Professional yet approachable. "Financial infrastructure, but make it friendly."
- **Logo:** Bold wordmark + gradient. Recognizable from a mile away.
- **Differentiator:** Best-in-class documentation design. Code + prose perfectly balanced.

### Visual Patterns This Audience Expects

1. **Dark mode by default** ‚Äî Light mode exists but dark is home
2. **Gradient accents** ‚Äî Not flat single-colors; subtle gradients on CTAs and highlights
3. **Generous spacing** ‚Äî No cramped UIs. Breathing room everywhere.
4. **Blur/glass effects** ‚Äî Frosted glass panels, blur backgrounds (macOS influence)
5. **Smooth animations** ‚Äî 200-300ms transitions. Spring physics. Nothing jarring.
6. **Monospace moments** ‚Äî Code snippets, terminal output, version numbers in mono
7. **Border radius: 8-12px** ‚Äî Rounded but not bubbly. Not 0, not 24px.
8. **Subtle borders** ‚Äî 1px borders in `rgba(255,255,255,0.1)` on dark backgrounds
9. **Status colors** ‚Äî Green = good, amber = warning, red = error (universal)
10. **Command palette** ‚Äî Cmd+K is expected. Fast keyboard access to everything.

## Recommended Brand Direction

### Name: **Clawhatch**
It's perfect. Don't change it. The closet/toolkit pun is clever without being forced, the claw reference is obvious, it's unique, memorable, short, and has strong domain availability potential.

### Colors: **Palette C ‚Äî "Twilight Reef"**
Dark hybrid with warm lobster red primary, twilight purple secondary, and electric teal accent. This positions Clawhatch as a premium developer tool (dark mode) that's friendlier than Vercel (color personality) and more focused than Railway (restrained palette).

### Logo: **Concept 5 ‚Äî "The Closet Door"**
A slightly ajar door with a claw handle or claw peeking out. It's the only concept that directly leverages the "Clawhatch" name pun. It's whimsical enough to be memorable, simple enough to work as a favicon, and tells a story: "open your Clawhatch."

### Typography: **Inter + JetBrains Mono + Space Grotesk**
Stay in the OpenClaw ecosystem. Add Space Grotesk for hero/marketing personality.

### Overall Vibe
> "The setup wizard that feels like opening a treasure chest."

**Not** cold and minimal like Vercel. **Not** chaotic-fun like Railway. Instead: **warm, competent, delightful.** Like a well-organized toolkit that makes you smile when you open it. Think: Linear's craft + Stripe's approachability + a lobster emoji.

**Tagline candidates:**
- "Open your Clawhatch" (primary)
- "Setup, snapped." 
- "Your AI assistant toolkit"
- "Everything you need. One snap."
